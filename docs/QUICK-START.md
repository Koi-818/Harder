# 快速开始指南

## 5 分钟快速上手

### 1. 安装依赖

```bash
npm install
```

### 2. 启动录制工具

```bash
npm start
```

### 3. 按照提示操作

- 输入要访问的网址
- 选择浏览器类型（Chromium/Firefox/WebKit）
- 指定 HAR 文件名（可使用默认值）

### 4. 进行浏览器操作

浏览器打开后，执行任何操作：
- 点击链接
- 填写表单
- 搜索内容
- 滚动页面

### 5. 关闭浏览器完成录制

只需关闭浏览器窗口，系统自动检测并完成录制。

## 输出位置

HAR 文件自动保存到 `hars/` 目录中。

## 常用命令

```bash
npm start          # 启动交互式录制
npm run record     # 同上
npm run dev        # 开发模式
npm test           # 运行测试
```

## 支持的浏览器

| 浏览器 | 命令选择 | 用途 |
|--------|--------|------|
| Chromium | 1 | 主流浏览器，推荐 |
| Firefox | 2 | 开源浏览器 |
| WebKit | 3 | Safari 引擎 |

## 录制提示

- 📝 所有网络请求都会被记录
- 💾 HAR 文件包含完整的请求/响应数据
- ⚡ 关闭浏览器窗口立即完成（无需按 Enter）
- 📊 录制完成后自动显示统计信息
